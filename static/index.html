چطور کار می‌کند؟

  1. متنت رو از هر منبعی (AI، یادداشت، وب) کپی کن.
  2. اون‌رو توی باکس «ورودی» پیست (paste) کن.
  3. با کلیک روی «پیش‌نمایش» قالب‌بندی Markdown رو ببین.
  4. روی «DOCX» بزن تا فایل Word آماده دانلود بشه.
